<h3>PROJECT DESCRIPTION :clipboard:</h3>
This is a simple Tic Tac Toe game developed in Java using the JSGE (JSimple Game Engine) library.

It was created as a practice project to improve my Object-Oriented Programming (OOP) skills and learn more about game development in Java.

---

<h3>TECHNOLOGIES USED :gear:</h3>

- Java
- JSGE (JSimple Game Engine)
- Object-Oriented Programming (OOP)

---

<h3>ABOUT JSGE :mag_right:</h3>

JSGE is a game engine for developing games and simulations in Java, created by my college professor David Buzatto.

Repository:
https://github.com/davidbuzatto/JSGE
