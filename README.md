<h1 align = "center">TIC TAC TOE</h1>
<br>
This is simple Tic Tac Toe game developed in Java using the JSGE (JSimple Game Engine) library.
It was created as a practice project to improve my Object-Oriented Programming (OOP) skills and learn more about game development in Java.

---

### Technologies Used

- Java
- JSGE (JSimple Game Engine)
- Object-Oriented Programming (OOP)

---

### About JSGE

JSGE is a simple game engine for developing games and simulations in Java, created by my college professor David Buzatto.

Repository:
https://github.com/davidbuzatto/JSGE

---

## How to Run

### Requirements

- Java JDK installed

### Clone the repository

```bash
git clone https://github.com/MariaClaraLisboa/tic-tac-toe-game.git
```

### Run the project

Open the project in your preferred IDE and execute the main class.
